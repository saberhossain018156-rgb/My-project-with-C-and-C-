# ATM Management System

A simple ATM Management System built using the C programming language.  
This project simulates basic ATM functionalities like account login, balance checking, deposit, withdrawal, and transaction management.

## Features
- User Login System
- Balance Check
- Deposit Money
- Withdraw Money


## Technologies Used
- C Programming Language


## How to Run
1. Compile the program:
   gcc atm.c -o atm

2. Run the executable:
   ./atm

## Learning Purpose
This project was created to practice:
- C programming fundamentals
- File handling
- Conditional statements and loops
- Function usage
- Struct concepts

## Future Improvements
- Password encryption
- Database integration
- GUI version
- Multiple user accounts
